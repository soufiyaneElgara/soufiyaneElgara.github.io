What is markdown#

  • light weight markup languge, with a plain text formating syntax.
  • can be converted into HTML/XHTML and other formating
  • it's main purpose is readability and ease of use

What is used for?#

  • readme files(Github etc)
  • forum & blog posts
  • used in many static site generetors

Markdown Editors:#

  • text editor extention like :VSCode, Atom, etc
  • for VSCode we have : auto-open markdown previw
  • MarkPad
  • HarooPad
  • Typora
